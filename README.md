# Subculture Tracker
게임을 플레이하면 가챠 결과를 자동으로 불러와 저장하며, 사용자는 프로그램 안에서 바로 기록을 확인할 수 있습니다.
복잡한 설정이나 수동 입력 없이 실행만 하면 바로 사용할 수 있도록 제작되었습니다.

## 프로그램 소개 
- 자동으로 원본 데이터를 추출하고, 공통 JSON 형식으로 저장
- 웹 기반 인터페이스를 데스크톱 창에서 바로 실행
- 브라우저 없이 간편하게 기록을 불러오고 관리

## 기능 소개
자동 기록 저장
- 가챠 결과를 자동으로 가져와 저장합니다. 매번 직접 캡처하거나 따로 적어둘 필요가 없습니다.

한곳에서 확인
- 저장된 기록을 프로그램 안에서 바로 확인할 수 있습니다. 게임별 결과도 쉽게 정리됩니다.

간편한 실행
- 설치 과정 없이 exe 파일만 실행해서 사용할 수 있습니다.

로그 기록
- 프로그램 실행 기록과 오류 내용은 자동으로 저장됩니다. 문제가 생겼을 때 확인하기 쉽습니다.

# 원리
대부분의 게임은 뽑기 기록을 확인할 수 있는 기능을 제공합니다.
일부 게임은 WebView를 사용해 게임 내부에서 기록 화면을 표시합니다.
게임은 실행 과정에서 여러 기록(Log)을 저장하며, 이 안에는 WebView가 열었던 링크 정보도 포함될 수 있습니다.
Subculture Tracker는 이 기록에서 관련 링크를 자동으로 찾아 접속한 뒤, 필요한 데이터를 수집합니다.
수집된 데이터는 프로그램 내부에서 보기 쉽게 정리되어 표시됩니다.

> WebView : 프로그램 안에 들어있는 작은 브라우저 기능입니다.
따로 인터넷 브라우저를 열지 않아도 앱 내부에서 화면을 바로 표시하고 사용할 수 있습니다.

# 제재 가능성에 대한 안내
> 이 프로그램은 게임 파일을 수정하거나 메모리를 조작하지 않으며, 게임이 저장한 기록(Log)에서 정보를 읽어오는 방식으로 동작합니다.
> 또한 게임에서 실제로 사용되는 기록 조회 링크를 이용해 데이터를 가져오기 때문에, 일반적인 사용만으로 제재 대상이 될 가능성은 낮다고 판단하고 있습니다.
> 개발 과정에서 잘못된 요청이나 짧은 시간 동안 여러 번의 요청 테스트를 진행했지만, 현재까지 계정 제한이나 경고 등의 문제는 확인되지 않았습니다.
다만 게임 운영 정책은 언제든 변경될 수 있으므로, 사용에 대한 최종 책임은 사용자 본인에게 있습니다.
